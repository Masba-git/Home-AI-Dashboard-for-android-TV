# Home AI Dashboard for Android TV 

A complete production-ready dashboard system for Android TV with 10+ integrated modules using **ONLY FREE APIS**.

## Features with Free APIs

- 🌤️ **Weather Module** - Uses wttr.in (completely free, no API key)
- 📰 **News Module** - Uses RSS feeds from BBC, CNN, Reuters (no API key)
- 📹 **CCTV Camera Integration** - Local RTSP/HTTP streams
- 🌐 **Network Status Scanner** - Local network scanning
- 📚 **Study Tasks Manager** - Local database storage
- 🤖 **AI Chatbot** - Rule-based assistant (no OpenAI costs)
- 🎥 **YouTube Study List** - Save and watch free educational content
- 📅 **Calendar Events** - Local event management
- 🕌 **Prayer Times** - Free Aladhan API (no key needed)
- 📝 **Notes System** - Local notes storage
- 👨‍👩‍👧‍👦 **Family Reminder Board** - Shared reminders

## NO API KEYS REQUIRED!

All modules work out of the box without any API keys or paid services.

## Prerequisites

- Python 3.8+
- pip (Python package manager)
- Android TV or any device with a browser
- Local network access
